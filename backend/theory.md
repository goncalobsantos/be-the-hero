## HTTP
 + GET --> ver/listar
 + POST --> alterar
 + PUT --> atualizar
 + DELETE --> apagar

## Parâmetros
 + 
 + Query params: Parametro para a rota após "?"
 +   - Filtros
 +   - Páginação
 +   - request.query --> { queryName: queryValue }
 + 
 + Route params: Parametro para identificar recurso (users/:id <--)
 + Request body: Corpo do request para criar ou alterar recursos (request.body)
